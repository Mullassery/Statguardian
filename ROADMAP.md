# StatGuard Roadmap

## Current Release: 0.1.0

### Completed ✓
- [x] Core schema validation (types, nullability, uniqueness)
- [x] Drift detection (PSI, Kolmogorov-Smirnov)
- [x] Anomaly detection (Z-score, IQR)
- [x] Multi-format support (Parquet, CSV, JSON, Arrow)
- [x] Lakehouse support (Delta Lake, Iceberg)
- [x] Declarative contract DSL
- [x] Python API (PyO3 bindings)

### Upcoming: 0.2.0
- [ ] Advanced statistical tests (Anderson-Darling, Shapiro-Wilk)
- [ ] Custom rule definitions in DSL
- [ ] Distributed execution (Spark, Dask)
- [ ] Data profiling enhancements
- [ ] Streaming validation support
- [ ] Web UI dashboard

### Future Roadmap
- [ ] dbt integration
- [ ] Apache Airflow operators
- [ ] Cloud data warehouse connectors (Snowflake, BigQuery, Redshift)
- [ ] Real-time alerting system
- [ ] Multi-tenancy support
